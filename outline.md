# План проекта Telegram-бота для записи к специалистам

## Цель проекта
Создание веб-интерфейса для управления Telegram-ботом, который позволяет специалистам автоматизировать запись клиентов через интеграцию с Google Calendar.

## Структура веб-сайта

### 1. Главная страница (index.html)
- **Hero секция**: Презентация сервиса с современным дизайном
- **Интерактивный календарь**: Отображение свободных/занятых слотов
- **Быстрая статистика**: Количество записей, доход, рейтинг
- **Карточки услуг**: Отображение доступных типов занятий
- **Отзывы клиентов**: Социальное доказательство эффективности

### 2. Страница настроек (settings.html)
- **Интеграция с Google Calendar**: 
  - Инструкции по получению API ключей
  - Форма подключения календаря
  - Настройка синхронизации
- **Настройка Telegram-бота**:
  - Инструкции по созданию бота через BotFather
  - Поле для ввода токена бота
  - Настройка вебхуков
- **Параметры бронирования**:
  - Рабочие часы и дни
  - Длительность слотов
  - Буферы между встречами
  - Часовой пояс (GMT+3)

### 3. Страница статистики (analytics.html)
- **Аналитика записей**: Графики популярности временных слотов
- **Финансовая статистика**: Доход по месяцам, средний чек
- **Клиентская база**: Повторные записи, предпочтения
- **Эффективность**: Конверсия запросов в записи

## Дизайн-концепция

### Визуальный стиль
- **Цветовая палитра**: Мягкие пастельные тона (светло-серый, бирюзовый, белый)
- **Типографика**: Современные безопасные шрифты (Inter, Roboto)
- **Иконография**: Минималистичные иконки из Feather Icons
- **Фон**: Чистый белый с акцентами на ключевых элементах

### Интерактивные элементы
- **Календарь**: Drag-and-drop для изменения расписания
- **Формы настройки**: Валидация в реальном времени
- **Графики**: Анимированные диаграммы с использованием ECharts
- **Модальные окна**: Для детальной информации о записях

## Технические компоненты

### Основные библиотеки
- **Anime.js**: Плавные анимации и transitions
- **ECharts.js**: Интерактивные графики и диаграммы
- **Splide.js**: Карусели для отзывов и услуг
- **p5.js**: Фоновые визуальные эффекты
- **Matter.js**: Физика для интерактивных элементов

### Функциональность
- **Реалистичные данные**: Примеры записей, статистики, отзывов
- **Адаптивный дизайн**: Оптимизация для всех устройств
- **Продуктовая фокусировка**: Все элементы работают без перезагрузки страницы

- **Прямая**: Специалисты, работающие индивидуально (тренеры, репетиторы, консультанты)
